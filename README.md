# Calculator Web Page

A simple calculator web page built using HTML, CSS, and JavaScript that performs basic arithmetic operations.

## 🚀 Features
- Addition
- Subtraction
- Multiplication
- Division
- Clean and user-friendly interface
- Responsive design for different screen sizes

## 🛠️ Technologies Used
- HTML
- Talwind CSS
- JavaScript
